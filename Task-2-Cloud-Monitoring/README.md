# Task 2: Cloud Monitoring and Alerts

## Objective
To set up monitoring and alerts for a cloud-based application.

## Implementation
AWS CloudWatch was used to monitor EC2 metrics such as CPU utilization and network traffic.

## Alerts
An alarm was configured to trigger when CPU utilization exceeds a threshold.

## Dashboard
A CloudWatch dashboard was created to visualize metrics.

## Tools Used
- AWS EC2
- AWS CloudWatch
